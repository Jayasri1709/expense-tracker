#  Expense Tracker

A GUI-based Expense Tracker built using Python and Tkinter.

##  Features
- Add, Delete, Update Expenses
- Search Function
- Monthly Report
- Total Expense Calculation
- Table Display (Treeview)

## Tech Used
- Python
- Tkinter
- CSV File Handling

## How to Run
1. Install Python
2. Run:
   python expense_gui.py
